# Logowanie

<img alt="obraz" width="290px" align="right" src="https://github.com/user-attachments/assets/a2241128-504c-48a7-a755-af97b0a9089a" />

Do panelu logujemy się poprzez **IP** kontrolera [**NASwebio**](../Kontroler/index.md). Interfejs jest dostępny z poziomu przeglądarki na dowolnym urządzeniu (PC, tablet, smartfon).
  
Aplikacja łączy w sobie funkcje, które zwykle są realizowane przez kilka osobnych urządzeń:

- kontroler kontroli dostępu (KD)
- centralę sygnalizacji włamania (SSWiN)
- moduł automatyki (np. HVAC)
- rejestrator zdarzeń powiązanych z obrazem
- system powiadomień i integracji zewnętrznych 

!!! success ""

    **Jedno** urządzenie zastępuje wiele systemów, upraszczając instalację i obniżając koszty.
