# Pliki skryptów

Panel zawiera listę **dostępnych plików skryptów** oraz umożliwia **tworzenie nowych**.

Skrypty wykonywane są w środowisku JavaScript zgodnym z **ECMAScript E5/E5.1**, z wybranymi elementami semantyki z **ES2015+**.

---
