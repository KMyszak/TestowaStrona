# Panel

- [**Pulpit**](Pulpit.md) - główny dashboard
- [**Urządzenie**](Urządzenie) - podstawowe informacje o kontrolerze
- [**Monitor zdarzeń**](Monitor-zdarzeń) - pełna lista zdarzeń z możliwością eksportu do pliku 
- **Kontrola dostępu** (kto, kiedy i gdzie ma wejście):

    - [**Użytkownicy**](Użytkownicy) - lista użytkowników
    - [**Karty**](Karty) - lista kart
    - [**Wzorce uprawnień**](Wzorce-uprawnień) - harmonogramy i warunki 
    - [**Przejścia/Czytniki**](Czytniki-(przejścia)) - lista czytników (przejść)

- **SSWiN** (system sygnalizacji włamania i napadu - monitorowanie czujników):

    - [**Strefy**](Strefy) - strefy do których przypisane są **wejścia** i **wyjścia** oraz ich statusy
    - [**Wejścia**](Wejścia) - lista **wejść** z możliwością edycji
    - [**Wyjścia przypisane do stref**](Wyjścia-(strefy)) - lista **wyjść** z możliwością edycji
    
- [**Wideorejestracja**](Wideorejestracja) - powiadomienia wideo (Campics) przy wystąpięniu zdarzenia
- [**Status (WEJ/WYJ)**](Status) - aktualny status **wejść** / **wyjść**

- **Ustawienia** - główne ustawienia kontrolera:

    - [**Pulpit**](Ustawienia-‐-Pulpit) - zarządzanie modułami panelu głównego
    - [**GSM**](Ustawienia-‐-GSM) - ustawienia GSM
    - [**Wejścia**](Ustawienia-‐-Wejścia) -  wybór modułu dla poszczególnych wejść
    - [**CMS**](Ustawienia-‐-CMS) - zarządzanie treściami internetowymi
    - [**Email**](Ustawienia-‐-Email) - ustawienia konta email      
    - [**Konfiguracja formatu kart**](Ustawienia-‐-Konfiguracja-formatu-kart) - edycja formatu dodanych kart      
    - [**Kamery**](Ustawienia-‐-Kamery) - zarządzanie podłączonymi kamerami
    - [**Campics**](Ustawienia-‐-Campics) - status wykorzystywanego urządzenia (pendrive)    
    - [**Powiadomienia**](Ustawienia-‐-Powiadomienia) - panel powiadomień      
    - [**Linki zewnętrzne**](Ustawienia-‐-Linki-zewnętrzne) - połączenie z innymi systemami bezpieczeństwa

- **Skrypt** (niestandardowe scenariusze projektowe):

    - [**Pliki skryptów**](Pliki-skryptów) - miejsce dodania nowego skryptu 
    - [**Filtry skryptów**](Filtry-skryptów) - lista skryptów 
    - [**Logi skryptów**](Logi-skryptów) - logi używanych skryptów 

- [**Admin panel**](Admin-panel) - panel administratora
